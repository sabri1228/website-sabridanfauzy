Selamat Datang

Ini adalah halaman utama dari materi pemrograman web.

PHP

PHP adalah bahasa pemrograman server-side yang digunakan untuk pengembangan web. Berikut adalah contoh kode PHP:

                <?php 

                echo "Hello, World!"; 

                ?>

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. Berikut adalah contoh kode JavaScript:

                console.log("Hello, World!");

CSS

CSS digunakan untuk mendesain tampilan halaman web. Berikut adalah contoh kode CSS:

                body { 

                    background-color: lightblue; 

                }

HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Berikut adalah contoh kode HTML:

                <!DOCTYPE html> 

                <html> 

                <head> 

                <title>Judul</title> 

                </head> 

                <body> 

                <h1>Hello World!</h1> 

                </body> 

                </html>
